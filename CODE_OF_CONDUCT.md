# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful, inclusive, and assume good intent.
- Disagree about technical decisions all you want; do not direct
  that disagreement at people.
- No harassment, no slurs, no personal attacks, no off-topic
  political flame wars.

## Reporting

If you experience or witness behavior that violates this code,
report it to **conduct@onplana.com**. Reports are confidential.
We commit to acknowledging within 2 business days and following
up on outcome.

Maintainers will:

1. Confirm receipt within 2 business days.
2. Investigate (talking to the parties involved as appropriate).
3. Decide on action — ranging from a private note through to PR
   / issue closure, repository ban, or escalation to GitHub
   abuse channels.
4. Communicate the outcome to the reporter.

## Enforcement scope

Applies to:

- This repository (issues, pull requests, discussions, code
  review comments)
- Any communication channels owned by Onplana that link from this
  repo (mailing lists, chat servers — none currently exist for
  the open-source packages, but we'll list them here if any spin
  up)

Does NOT apply to:

- Private conversations or third-party platforms not affiliated
  with this project
- The closed Onplana product's customer support channels — those
  have their own conduct expectations

## Why we publish this

Because the MCP ecosystem is small and the standards we set early
matter. We'd rather state expectations explicitly + boringly than
discover later that we needed them and didn't have them in
writing.
